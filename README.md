# Fair-Resume-Screening-System (Unbiased Hiring AI)

This is a complete full-stack web application designed to remove bias from the hiring process.

## 📁 Project Structure

- **/backend**: Node.js + Express API server.
- **/frontend**: React.js client application.
- **/database**: Seeding scripts for sample data.
- **/cloud-functions**: Serverless code for GCP integration.
- **/docs**: Detailed documentation and README.

## ⚡ Quick Start

### 1. Install All Dependencies
```bash
npm run install-all
```

### 2. Configure Environment
Create a `.env` file in `backend/` (Refer to `backend/.env.example`).

### 3. Seed Sample Data
```bash
node database/seed.js
```

### 4. Run the Project
```bash
npm start
```

For more details, please refer to the documentation in `/docs/README.md`.
